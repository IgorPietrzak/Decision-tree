Decision tree implementation¶

In [39]:
def process_entry(entry,petal_length_threshold, petal_width_threshold):
    if entry[2] <= petal_length_threshold:
        return 0
    elif entry[3] <= petal_width_threshold:
        return 1
    else:
        return 2
    
def run_model(data,petal_length_threshold, petal_width_threshold):
    output = []
    for entry in data:
        output.append(process_entry(entry,petal_length_threshold, petal_width_threshold))
    return output

Calculate misclassification rate¶

In [40]:
def get_misclassification_rate(training_output, training_labels):
    count = 0
    for i in range(len(training_output)):
        if training_output[i] == training_labels[i]:
            count = count + 1
            i = i + 1
    return count/len(training_output)

Primitive way of determining model parameters¶

From all possible pair plots (considering the diagonal to check just one feature at a time) we see that petal length and petal width features are the best at distinguishing the flowers. This is why they are used in the model.¶

Since I don't know how to determine model parameters yet, we take the primitive approach of guessing parameters and replacing them if the model scores better.¶
